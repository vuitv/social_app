## Getting Started 🚀

### Install Flutter SDK
```sh
git clone https://github.com/flutter/flutter.git --depth 1 -b 3.22.2 $HOME/flutter
export PATH="$PATH:$HOME/flutter/bin"
flutter doctor -v
```

### Run project

- Get dependencies
```sh
flutter pub get
```

- Run
```sh
flutter run
```
